# Healthcare Power BI Dashboard

## Project Overview
This project presents an interactive Power BI dashboard analyzing hospital bed availability and utilization across counties.  
The goal is to identify healthcare capacity patterns, regional resource distribution, and potential pressure on hospital systems.

## Dataset
The dataset includes hospital-level information such as:

- Facility name  
- County  
- Available beds (BED_AVL)  
- Licensed beds (BED_LIC)  
- Operational days and reporting metrics  

These fields enable analysis of capacity, utilization rates, and regional healthcare demand.

## Dashboard Features
The Power BI report contains:

- Total available beds by county  
- Total licensed beds by county  
- Bed utilization percentage by county  
- Overall utilization KPI indicator  

The visuals help highlight:

- Counties with the highest healthcare capacity  
- Regions operating near full utilization  
- Potential imbalance in healthcare resource distribution  

## Key Insights
- Large metropolitan counties dominate total bed availability.  
- Several counties operate close to full utilization, indicating pressure during demand spikes.  
- Resource distribution varies significantly across regions.
